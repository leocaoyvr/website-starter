#   Staic Site Boilerplate with Pug Sass Gulp

##  Technology Stack

**Language**

-   PUG (HTML)
-   SASS (CSS)
-   jQuery (Javascript)

**Tools**

-   Gulp & Plugins
-   Browser Sync
-   Sublime Text

##  Project Specs

-   Languages: Chinese and English
-   Breakpoints: 6

##  Install & Setup

**Option 1**

1.  Upload *dist* to server
2.  Configure */dist/en/index.html* or */dist/cn/index.html* as default page

**Option 2**

1.  `npm install`
2.  `gulp`
3.  Upload the latest *dist* to server
